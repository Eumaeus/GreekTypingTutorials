# Typing Polytonic Ancient Greek on macOS

**Using Keyman + GreekClassical Keyboard**

This tutorial assumes you know the Greek alphabet and what breathings, accents, iota subscript, etc. look like. It assumes **no prior experience** installing programs or changing keyboard settings on a Mac.

## Step 1: Install Keyman (the keyboard program)

1. Open Safari (or your usual web browser) and go to:  
   **<https://keyman.com/mac/>**

2. Click the large **Download Keyman for macOS** button.

3. When the download finishes, find the file (it ends in `.dmg`) in your **Downloads** folder and **double-click** it.

4. A new window will open. **Double-click** the **Install Keyman** icon inside that window.

5. Follow the on-screen prompts and click **Install** when asked.

6. Keyman will finish installing. You can close the installer window when it’s done.

**Important macOS Security Note**  
On newer versions of macOS you may see a warning that the app is from an unidentified developer or that it needs permissions.  
- Go to **System Settings → Privacy & Security**.  
- Scroll down and click **Allow** or **Open Anyway** next to the Keyman message.  
- You may also be asked to allow Keyman to control your computer or monitor input. **Click Allow** — this is required for the keyboard to work.

After installation you should see a small Keyman icon in the top-right menu bar (near the clock, Wi-Fi, and battery icons).

## Step 2: Install the GreekClassical Keyboard

1. In your browser, go to:  
   **<https://keyman.com/keyboards/greekclassical>**

2. Click the **Install keyboard** button for **macOS**.

3. Keyman should open and install the keyboard automatically.

If the direct button doesn’t work:
- Download the `.kmp` file from the page.
- Double-click the `.kmp` file, **or**
- Click the Keyman icon in the menu bar → **Configuration** → **Keyboard Layouts** tab → **Add** and select the downloaded `.kmp` file.

## Step 3: Switch to the Greek Keyboard

1. Look at the top-right corner of your screen in the menu bar. You will see an icon that shows the current keyboard (usually something like **ABC**, a flag, or the word “English”).

2. **Click** that icon.

3. In the menu that appears, choose **Keyman** → **Greek Classical** (the exact name may vary slightly).

You can now start typing in Greek in any program (Pages, Word, Google Docs, Notes, etc.).

To switch back to English:
- Click the same menu-bar icon again and choose **English** or **ABC**.

**Tip:** You only need to do this once per session. macOS will usually remember your last choice.

## Step 4: Typing Greek Letters

The layout is **mnemonic** — most letters are on keys that sound or look similar to English.

### Common Letter Mappings (lowercase)
- `a` → α
- `b` → β
- `g` → γ
- `d` → δ
- `e` → ε
- `z` → ζ
- `h` → η
- `q` → θ
- `i` → ι
- `k` → κ
- `l` → λ
- `m` → μ
- `n` or `v` → ν
- `x` → ξ (or `j`/`c` in some cases)
- `o` → ο
- `p` → π
- `r` → ρ
- `s` → σ (becomes ς automatically at end of word)
- `t` → τ
- `u` → υ
- `f` → φ
- `y` → ψ
- `w` → ω

**Capitals**: Hold **Shift** (e.g., `Shift + w` → Ω)

**To type a normal Latin letter** (e.g., in English words or names): Type the **backtick** `` ` `` first, then the letter.  
Example: `` `a `` → a (Latin)

## Step 5: Adding Diacritical Marks

**Type the vowel first**, then the diacritic key(s). Order usually doesn’t matter.

### Diacritic Keys
| Mark                  | Key | Name                  |
|-----------------------|-----|-----------------------|
| ῾ (rough breathing)   | [   | dasia                 |
| ᾿ (smooth breathing)  | ]   | psili                 |
| ι (iota subscript)    | \|  | ypogegrammeni         |
| ¨ (diaeresis)         | +   | dialytika             |
| ` (grave)             | \   | varia                 |
| ´ (acute)             | /   | oxia                  |
| ˜ (circumflex)        | =   | perispomeni           |
| ˘ (breve)             | -   | vrachy                |
| ¯ (macron)            | _   | macron                |

### Examples
- `a /` → ά
- `h [ /` → ἥ
- `w = |` → ῷ
- `u + /` → ΰ
- `a [ = |` → ᾇ (rough + circumflex + iota)

**Sigma** automatically becomes final ς at the end of a word.

**Full official chart** (very useful while learning):  
https://help.keyman.com/keyboard/greekclassical

## Step 6: Practice & Tips

1. Open **Notes**, **Pages**, or **TextEdit**.
2. Switch to the Greek keyboard using the menu bar.
3. Type: `q e / o s` → θεός
4. Type: `o i [ = o n` → οἷον
5. Try combining breathings and accents.

**Tips**
- Diacritics only attach to vowels (and sometimes ρ).
- If a combination doesn’t look right, just keep typing — it often corrects itself.
- You can re-type a diacritic key to remove or adjust it in many cases.
- Keyman has an on-screen keyboard you can open from the menu bar icon if you get stuck.
