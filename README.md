# Greek Typing Tutorials

Easy-to-follow tutorials for typing polytonic Ancient Greek using the **Keyman GreekClassical** keyboard on Windows and macOS.

These tutorials are designed for students in Introductory Ancient Greek who already know the alphabet and diacritical marks but may have little or no experience changing computer settings or installing software.

## Why Keyman GreekClassical?
- Same keyboard and typing experience on both Windows and macOS
- Mnemonic layout (easy to remember)
- Diacritics typed *after* the vowel
- Automatic final sigma (ς)
- Works in Word, Google Docs, LibreOffice, etc.
- Free and works offline after installation
